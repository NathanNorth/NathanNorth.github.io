SPYCAM

/leaderboards:

Gets leaderboards from all regions, and lets users browse to a specific point in the leaderboard. This command is fully functional; the discord bot is actually making the api calls.

/opt-in:

Allows a user to opt into sharing their data with Spycam. Note how this message is ephemeral: nobody can see the it except the user it's directed act.

/profile:

Shows key stats for a player's last ten games and specific info from their last three. By default it shows your own profile but you can optionally look up anyone who's opted-in.

Rank Up Notifications:

After a server designates a channel for announcements, the bot would send simple messages like these to congratulate players on their ranked achievements. I'm deliberately keeping these messages simple: its not about the bot showering you in confetti, the messages should be a catalyst for your discord buddies to give you a proper celebration.
