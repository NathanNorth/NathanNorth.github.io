vc-notification-bot (github):

The vc-notification-bot is a dead simple tool to help keep track of when people join voice channels in Discord using the brand new Slash Command API. The bot is built reactively using Discord4J and a R2DBC managed PostgreSQL database backend. The project is currently at a beta release of version 1.1.

Nathan's AI Bot (closed source):

The AI bot is a Discord bot built using GPT-2 (namely gpt-2-simple/aitextgen) and Discord4J to create AI conversations trained off of messages sent in your Discord servers. The bot is currently in closed beta and seeing active developement from yours truly. The latest stable release is version 0.1.6, but the bot in online online intermittently in select servers.
